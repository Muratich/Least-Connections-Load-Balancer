# Least-Connections-Load-Balancer

## How to run:

Run simulator/test.py that will run three backend servers, load balancer and simulator.
If some ports are busy in your machine, you can change them in the script to free ones (e.g BACKEND_3_HTTP from 8083 to 8085).

### Frontend access

After running scripts, you can access the GUI via the address:
http://127.0.0.1:8100/

## Requirements:

Install following:
Python 3.14.3
Go 1.25.6
